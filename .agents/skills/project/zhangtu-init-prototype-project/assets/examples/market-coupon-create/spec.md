# 新建优惠券活动

## 场景
用于营销运营在评审阶段快速配置一个优惠券活动，覆盖券模板、面额、门槛、目标客群和投放渠道。

## 页面结构
- 顶部展示活动名称、核心说明和摘要预览。
- 左侧提供完整的活动配置表单。
- 右侧展示模板库与评审清单，帮助运营快速核对规则。

## 交互要求
- 修改表单后，右侧预览卡片实时刷新。
- 切换券模板时，模板标签和预览状态同步更新。
- 保存草稿和提交评审按钮保留在页面中，作为流程占位操作。

## 验收标准
- 页面可独立打开并通过 `createRoot(...).render(...)` 启动。
- `index.html`、`index.tsx` 和 `spec.md` 必须同时存在。
- 示例页应保留 `screenshot.png` 与 `assets/images/avatar-premium.svg` 作为项目资源。
