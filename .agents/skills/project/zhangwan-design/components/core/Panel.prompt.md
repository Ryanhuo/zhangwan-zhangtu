带标题的描边面板，对应 `.card-box`——浅色标题条覆盖在带边框的白色主体之上，
用于在页面里对一组相关内容分组（筛选行、标签列表、用户画像卡片等）。

```jsx
<Panel title="用户标签">
  <div>...内容...</div>
</Panel>
```

用法：`title` 是标题条里的纯文字/节点，不带图标或右侧操作区——如果需要"更多"
链接或操作按钮，把它们作为 `children` 内部自己的一行来做，不要指望 Panel 提供
标题区插槽。`children` 内边距固定 20px，长内容（长列表、宽表格）会撑高面板而
不滚动，需要滚动请在 `children` 内部自己包一层 `overflow: auto`。

搭配惯例：Panel 常与 SectionTitle 配合使用——页面级分区用 SectionTitle（竖线
标题），分区内的卡片分组用 Panel（描边标题条）。多个 Panel 并排时用 grid/flex
+ gap 布局，不要互相挤在一起。

边界情况：`title` 留空时标题条仍占据 39px 高度（不会塌陷），避免相邻 Panel
高度不一致；如果确实不需要标题条，应该用别的容器而不是 Panel。
