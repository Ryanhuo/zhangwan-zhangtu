# 3.x.5「业务规则」模板

每个功能模块的 **`3.x.5 业务规则`** 描述**本模块特有**的排序、搜索、加载、交互、校验、删除等行为；与 **2.3 全局说明** 重复的内容写 **`遵循 2.3 全局说明`**，不在此重复展开。

HTML 标题示例：

```html
<h4 id="3255-业务规则">3.2.5 业务规则</h4>
```

---

## 六类结构（有则写、无则省略）

| 小节 | 何时写 |
|------|--------|
| **排序规则** | 本模块列表/树的新增排序、默认排序与全局不一致时 |
| **搜索规则** | 有搜索框、筛选、分组展开逻辑时 |
| **加载规则** | 页面初始布局、分区默认态、懒加载详情时 |
| **交互规则** | 悬停按钮、折叠拉伸、右键/操作菜单、删除前置条件等 |
| **校验规则** | 本模块特有校验（必填、唯一性等）；通用必填见 §1.3 |
| **删除约束** | 不可删条件、关联校验（可并入交互规则最后一条） |

## 完整示例：核算架构（可复制）

以下为 **`3.x.5 业务规则`** 标准示例；生成其他模块时替换页面名、区域名、按钮名，**保留六类结构与编号写法**。

### 排序规则

(1) 新增的体系或单位，按新增时间的先后顺序，**升序**排列显示。即 **先增加的在最上方** 显示。

> 若与 2.3 全局说明 (9)② 一致，可写：`遵循 2.3 全局说明（列表新增排序）`。

### 搜索规则

(1) 搜索框支持 **模糊查询**，字母 **不区分大小写**。

(2) 按 **【回车键】** 或点击 **【查询】** 按钮，根据搜索框输入的内容，查询单位；相关结果 **按分组展开** 显示。

### 加载规则

(1) 进入 **【核算架构】** 页面，左侧体系分组布局，默认展示全部体系分组，分组 **全部折叠**；点击分组再展开当前分组下的单位。

(2) 进入 **【核算架构】** 页面，右侧布局 **默认为空**；点击左侧单位，右侧布局再显示当前单位的 **【基本信息】**、**【账号信息】**、**【税号】**。

### 交互规则

(1) 鼠标 **移入** 已建的体系分组时，显示 **【编辑】** 及 **【删除】** 按钮。

(2) 鼠标 **移出** 已建的体系分组时，隐藏 **【编辑】** 及 **【删除】** 按钮。

(3) 点击已建单位的 **【操作】** 时，显示 **【增加子项】** 及 **【删除】** 按钮。

(4) 已被单据、相关配置关联使用当前单位作为数据源，或者单据提交过流程的，均 **无法被删除**。

(5) 页面被折叠后，右侧列表 **自动拉伸** 显示。

### 校验规则（本模块补充）

(1) 必填项以标签旁红色 `*` 标识；点击 **【保存】** 时校验，错误在字段下方展示（见 `prd-html-conventions.md` §1.3）。

(2) 字段说明「说明」列含 **「唯一」** 的字段，【保存】时重复提示并阻断（`遵循 2.3 全局说明（唯一性）`）。

### 删除约束（可选独立小节）

若交互规则 (4) 已写清，**删除约束** 小节可省略；复杂场景可单独列出：

- 已被单据引用 → 禁止删除，提示「该单位已被单据引用，无法删除」
- 已提交流程 → 禁止删除，提示「该单位已关联流程，无法删除」

## Markdown 写法（Agent 生成 MD 源稿）

```markdown
#### 3.2.5 业务规则

**排序规则**
(1) 新增的体系或单位，按新增时间的先后顺序，升序排列显示。即先增加的在最上方显示。

**搜索规则**
(1) 搜索框支持模糊查询，字母不区分大小写。
(2) 按【回车键】或点击【查询】按钮，根据搜索框输入的内容，查询单位。相关结果按分组展开显示。

**加载规则**
(1) 进入【核算架构】页面，左侧体系分组布局，默认展示全部体系分组，分组全部折叠。点击分组再展开当前分组下的单位。
(2) 进入【核算架构】页面，右侧布局默认为空。点击左侧单位，右侧布局再显示当前单位的【基本信息】、【账号信息】、【税号】。

**交互规则**
(1) 鼠标移入已建的体系分组时，显示【编辑】及【删除】按钮。
(2) 鼠标移出已建的体系分组时，隐藏【编辑】及【删除】按钮。
(3) 点击已建单位的【操作】时，显示【增加子项】及【删除】按钮。
(4) 已被单据、相关配置关联使用当前单位作为数据源或者单据提交过流程的，均无法被删除。
(5) 页面被折叠后，右侧列表自动拉伸显示。

**校验规则**
(1) 必填项校验遵循字段说明表「必填提示」段；点击【保存】时触发。
(2) 唯一性字段遵循 2.3 全局说明。
```

## 与 2.3 全局说明的分工

| 内容 | 写在哪里 |
|------|----------|
| 全站列表默认编码升序、分页 20 条、下拉【加载更多】 | **2.3 全局说明** |
| 本页左侧树折叠、右侧空态、悬停出按钮 | **3.x.5 业务规则** |
| 本页搜索按分组展开 | **3.x.5 搜索规则** |
| 编码仅字母数字下划线 | **2.3**；模块内写「遵循 2.3」 |
| 本模块删除关联条件 | **3.x.5 交互规则** 或 **删除约束** |
