# 掌图原型工具 — 同事安装指南

掌图（zhangtu）是一套本地原型管理工具，支持页面预览、需求标注、版本管理和分享链接发布，全部在本机运行，无需联网账号。

---

## 前置条件

安装前确认以下软件已就绪：

| 软件 | 最低版本 | 检测命令 |
|------|---------|---------|
| Node.js | 18.x 以上 | `node -v` |
| Git | 任意版本 | `git --version` |
| Claude Code | 最新版 | `claude --version` |

- Node.js 官网：https://nodejs.org（选 LTS 版本）
- Git 官网：https://git-scm.com
- Claude Code：https://claude.ai/code

---

## 方式一：用 AI 助手一句话安装（推荐）

### 第一步：安装掌图 Skill（一次性）

在终端运行以下命令，把掌图安装助手注册为 Claude Code 的全局技能：

**Mac / Linux：**
```bash
mkdir -p ~/.claude/skills/zhangtu-installer && curl -fsSL https://raw.githubusercontent.com/Ryanhuo/zhangwan-zhangtu/main/.agents/skills/project/zhangtu-installer/SKILL.md -o ~/.claude/skills/zhangtu-installer/SKILL.md
```

**Windows（PowerShell）：**
```powershell
New-Item -ItemType Directory -Force -Path "$env:USERPROFILE\.claude\skills\zhangtu-installer" | Out-Null
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/Ryanhuo/zhangwan-zhangtu/main/.agents/skills/project/zhangtu-installer/SKILL.md" -OutFile "$env:USERPROFILE\.claude\skills\zhangtu-installer\SKILL.md"
```

这只需要做**一次**，之后在任何项目都可以使用。

---

### 第二步：用 Claude Code 安装工作区

1. 打开 Claude Code，进入你想放原型项目的文件夹（比如桌面）
2. 直接说：**「我要在这里安装掌图」**
3. Claude 会自动问你项目名称，然后完成安装、启动预览

就这么简单。

## 方式二：手动命令安装

如果不使用 Claude Code，也可以直接在终端操作：

### 第一步：创建工作区

```bash
# 进入你想放项目的目录（例如桌面）
cd ~/Desktop

# 从 GitHub 初始化掌图工作区（将 my-project 替换为你的项目名）
npx github:Ryanhuo/zhangwan-zhangtu init my-project
```

### 第二步：安装依赖

```bash
cd my-project
npm install
```

等待安装完成（首次大约 1–2 分钟）。

### 第三步：启动预览

```bash
npm start
```

浏览器会自动打开预览页面（端口 6320）。如果没有自动打开，手动访问：

```
http://127.0.0.1:6320
```

## 基本使用

### 添加页面

在 `src/pages/` 下新建文件夹，按以下结构添加文件：

```
src/pages/我的页面/
├── index.html      # 必须
├── index.tsx       # 必须
└── spec.md         # 可选，页面说明（第一行 # 标题会作为页面名）
```

保存后预览 Shell 会自动刷新，左侧导航出现新页面。

### 创建版本

在预览 Shell 右上角点击「版本管理」→「+ 新建版本」，填写版本名和说明，勾选要包含的页面，保存即可。

### 预览版本

在版本列表点击「预览」，新标签页打开只读版本预览，可分享给团队评审。

## 常见问题

**Q: 端口被占用了怎么办？**
```bash
npm start -- --port 6321
```

**Q: 页面没出现在导航栏？**
运行 `npm run check:pages` 查看具体诊断信息。

**Q: npm install 很慢？**
切换国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

**Q: 如何拿到分享链接？**
需要先配置发布 Token（联系系统管理员获取），然后在版本管理里点「发布」。

**Q: 如何更新掌图到最新版本？**
```bash
npm update zhangwan-zhangtu
npm install
```
