# 设计系统浏览器 PRD

## 页面背景与目标

`design-system` 页面是掌图内置的设计系统浏览器，用于集中展示 `.agents/skills/project/zhangwan-design/` 的真实设计资产，帮助产品、设计和研发快速理解掌玩后台的视觉基线与组件规范。页面目标是把设计令牌、规范卡片、组件示例和完整 UI Kit 统一收口到一个可浏览、可预览、可核对的入口，并确保展示内容与设计系统源文件保持同步。

## 核心功能清单

- 展示 zhangwan-design 的 guidelines 规范卡片，覆盖品牌、色彩、字体、间距等基础规则。
- 展示组件示例卡片，帮助浏览常用内容区组件、数据组件、表单组件和反馈组件的真实效果。
- 提供完整 UI Kit 全页预览，便于查看整套后台界面的组合关系与视觉气质。
- 通过 iframe 直接渲染 `assets/zhangwan-design/` 下的真实 HTML 资产，而不是手抄截图或重写示例。
- 提供完整 Token 列表索引，便于按颜色、字体、间距、圆角、阴影等维度查阅规范明细。

## 关键交互说明

- 用户可通过左侧固定导航快速跳转到概览、品牌、色彩、字体、间距、组件、UI Kit 与 Token 列表等分区。
- 每张规范卡片或组件卡片在内容区以内嵌 iframe 方式展示，用户可直接看到真实渲染结果与内置交互。
- UI Kit 区域支持整页预览，用于连续查看完整后台界面结构，而不是仅看单一组件。
- Token 列表以表格形式呈现，用户可按分组快速扫描命名、用途与具体值。

## 验收标准

- 页面能清晰展示 zhangwan-design 的令牌、规范卡片、组件示例与 UI Kit 四类核心资产。
- 所有展示内容均来自 `assets/zhangwan-design/` 的真实镜像文件，不能出现手工抄录导致的内容漂移。
- guidelines、组件示例和 UI Kit 均可在页面内正常预览，iframe 加载无空白或明显报错。
- 页面信息结构清晰，用户可通过导航在各分区间快速定位目标内容。
- Token 列表分类完整，可支持设计、产品和研发核对关键设计变量。
