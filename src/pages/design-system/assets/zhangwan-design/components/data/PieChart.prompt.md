环形图，对应 `components/chartsNew/pieChart.vue`（基于 @antv/g2 的 theta 坐标
系，`innerRadius: 0.6` / `radius: 0.75`），用于展示占比构成（渠道占比、类型
分布等）。

```jsx
<PieChart
  data={[{ label: '巨量引擎', value: 62 }, { label: '磁力引擎', value: 38 }]}
  formatValue={(v) => `${v} 次`}
/>
```

视觉规则：逐字复用源码里的分类调色板（23 色循环），右侧是纵向图例（名称 +
占比百分比，不是数值），悬停扇区或图例项都会互相联动高亮，其余项降到 0.35
透明度，并弹出提示框（名称/数值/占比）。

用法：`data` 里每项可选 `color` 覆盖调色板取色；不传时按数组顺序从调色板
取色，因此调整 `data` 顺序会连带改变配色，跨次渲染保持顺序稳定很重要。
`showLabelText` 用于始终显示"名称：数值(占比%)"文案，而不是仅悬停显示——
仅在图表需要脱离交互静态展示时打开（如导出截图场景）。

边界情况：`data` 为空或所有 `value` 总和为 0 时显示"暂无数据"，不会渲染
空圆环；图例项过多（超过 6-8 项）会在 160px 宽的图例列里挤得很密，建议超过
这个数量时改用 DataTable 展示明细，图表只做 Top N + "其他"聚合。
